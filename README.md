# PDF合并工具

一个基于Python和pywebview的PDF合并工具，支持普通合并和发票合并（两张发票一页）模式。

## 功能特点

- **普通合并模式**：将多个PDF文件按顺序合并成一个文件
- **发票合并模式**：将多个发票PDF合并，每页显示两张发票（上下排列）
- **可视化界面**：直观的工具栏、文件列表、预览区域和状态栏
- **实时预览**：点击文件列表中的PDF可预览内容
- **进度显示**：合并过程中显示进度条

## 安装

1. 安装Python 3.7或更高版本
2. 安装依赖包：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行程序：

```bash
python pdfm_v1.py
或者
python pdfm_v2.py
```

注意：pdfm_v1.py是版本1拥有完善功能，但对于复杂的页面顺序调整，或者多页pdf中的调整不行，但是界面简单
pfm_v2.py 拼接调整删除增加功能非常完善，但是总感觉没那么美观，可能是初恋是最美好的原因。另外gemini3 比国内的大模型强很多。即使不用cursor之类的就用它chat界面也非常好
v3增加了卡片旋转，多卡片操作。多页面预览什么的，并且做了逐步清晰的做法。即使再大的图片也能正常处理。


2. 点击"选择PDF文件"按钮，选择要合并的PDF文件

3. 选择合并模式：

   - 普通合并：按顺序合并所有PDF
   - 发票合并：每页显示两张发票
4. 点击"开始合并"按钮，选择输出文件位置
5. 等待合并完成

## 注意事项

- 发票合并模式下，程序会自动将两张发票调整为适合A4页面的大小
- 预览功能显示PDF的第一页内容
- 合并后的文件将按照选择的顺序进行合并

## 依赖说明

- `pywebview`：用于创建桌面应用界面
- `PyPDF2`：用于PDF文件操作
- `Pillow`：用于图像处理
- `reportlab`：用于创建PDF文件
- `PyMuPDF`：用于PDF渲染和预览（可选，提供更好的预览效果）
